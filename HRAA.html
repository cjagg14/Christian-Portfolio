Hybrid Rocket Analysis App

Project Description

While working on the MetRocketry design team (one of TMU's rocketry design team), I was tasked with modeling how the combustion chamber and other properties would effect the outcome of a hybrid rocket. I then took it upon myself, creating an application that can be use to model a hybrid rocket. Using HRAP (another application found on the internet) I started taking in some ideas on how I want my application to function.

After a lot of trial and error, I have created the application seen above. This application uses different properties found from the hybrid rocket and a few other variables allowing for a rough estimate on the thrust, specific impulse, and regression rate. This app can also produce many different kinds of graphs by allowing the user to change the x and y plots to what they desire. This is demonstrated in the image below.

Although the results aren't 100% accurate they still provide a rough estimate of what the rocket will do. This is useful as you can quickly see how different parameters change the outcome of the rocket.
