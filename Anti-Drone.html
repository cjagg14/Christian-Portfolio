Anti-Drone Project

Project Description

In my fourth year of study, I undertook a project that integrated the use of an Arduino kit alongside a 3D printer to engineer a specialized device. The primary objective of this endeavor was to develop an attachment for a drone, designed specifically to intercept and capture drones operating within restricted airspace. This project not only honed my technical skills in programming and 3D modeling but also deepened my understanding in drone technology and its applications in maintaining security.
